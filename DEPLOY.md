# 🚀 5분 안에 배포하기

## 준비물
- GitHub 계정
- index.html 파일
- README.md 파일

## 단계별 가이드

### 1️⃣ 리포지토리 만들기 (1분)
```
1. github.com 접속
2. 오른쪽 상단 + 버튼 → New repository
3. 이름: daily-tracker
4. Public 선택
5. Create repository 클릭
```

### 2️⃣ 파일 업로드하기 (2분)
```
1. "uploading an existing file" 클릭
2. index.html과 README.md를 드래그
3. Commit changes 클릭
```

### 3️⃣ GitHub Pages 켜기 (1분)
```
1. Settings 탭
2. 왼쪽 Pages 클릭
3. Branch: main 선택
4. Save 클릭
```

### 4️⃣ 접속하기 (1분)
```
몇 분 기다린 후...
https://your-username.github.io/daily-tracker
```

## 🆘 문제 해결

**404 에러가 나요**
→ 파일 이름이 `index.html`인지 확인
→ 5분 정도 기다려보세요

**페이지가 안 뜨요**
→ Settings > Pages에서 활성화되었는지 확인
→ Branch가 main으로 선택되었는지 확인

**휴대폰에서 안 열려요**
→ URL 주소를 정확히 입력했는지 확인
→ https://를 빼먹지 않았는지 확인
