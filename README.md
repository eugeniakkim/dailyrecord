# 일상 기록 관리 📊

타이드와 누렁이의 일상 활동을 기록하는 웹 애플리케이션입니다.

## 🚀 GitHub Pages 배포 방법

### 1단계: 리포지토리 생성
1. [GitHub](https://github.com)에 로그인
2. 오른쪽 상단 `+` 버튼 → `New repository` 클릭
3. Repository name: `daily-tracker` (또는 원하는 이름)
4. **Public** 선택
5. `Create repository` 클릭

### 2단계: 파일 업로드
1. 생성된 리포지토리 페이지에서 `uploading an existing file` 클릭
2. `index.html` 파일을 드래그 앤 드롭
3. `Commit changes` 클릭

### 3단계: GitHub Pages 활성화
1. 리포지토리 페이지에서 `Settings` 탭 클릭
2. 왼쪽 메뉴에서 `Pages` 클릭
3. **Source** 섹션에서:
   - Branch: `main` 선택
   - Folder: `/ (root)` 선택
4. `Save` 클릭

### 4단계: 접속하기
- 몇 분 후 페이지 상단에 URL이 표시됩니다
- `https://your-username.github.io/daily-tracker`
- 이 URL을 휴대폰 브라우저에서 접속하세요!

## 📱 휴대폰에서 사용하기

## 🎯 주요 기능

### 활동 기록
- 🏃 운동: 종류, 시간 기록
- 🌙 야식: 음식, 칼로리 기록
- 💊 소화제: 제품명, 복용 시간 기록
- 🏠 청약: 아파트명, 평형, 가격, 발표일 기록

### 데이터 관리
- 📥 **CSV 내보내기**: 데이터를 CSV 파일로 저장
- 📤 **CSV 가져오기**: 저장된 CSV 파일 불러오기
- 🖱️ **드래그 앤 드롭**: CSV 파일을 드래그해서 업로드

### 보기 옵션
- 📊 대시보드: 통계 및 최근 활동
- 📅 달력: 월간 보기
- 📆 1주일 보기: 주간 달력 형식
- 🔍 활동별 페이지: 운동, 야식, 소화제, 청약

## 💾 데이터 저장 방식

- 데이터는 **브라우저의 LocalStorage**에 저장됩니다
- 각 기기/브라우저마다 독립적으로 저장됩니다
- **중요**: 브라우저 캐시를 지우면 데이터가 삭제될 수 있으니 정기적으로 CSV로 백업하세요!

## 🔄 기기 간 데이터 동기화

### 방법 1: CSV 백업/복원
1. 휴대폰에서 `CSV로 저장` 클릭
2. 이메일이나 클라우드에 저장
3. 다른 기기에서 `CSV 불러오기`로 복원

### 방법 2: 정기 백업 (권장)
- 매주 한 번씩 CSV로 저장하여 백업
- Google Drive나 이메일에 저장

## 🛠️ 문제 해결

### 데이터가 사라졌어요
- 브라우저 캐시가 지워졌을 가능성
- 백업한 CSV 파일이 있다면 불러오기로 복원

### 휴대폰에서 안 열려요
- URL을 정확히 입력했는지 확인
- GitHub Pages가 활성화되었는지 확인 (Settings > Pages)
- 배포 후 몇 분 정도 기다려주세요

### CSV 불러오기가 안 돼요
- CSV 파일 형식이 올바른지 확인
- 이 앱에서 내보낸 CSV 파일만 호환됩니다

## 📝 업데이트 방법

파일을 수정하고 싶다면:
1. GitHub 리포지토리에서 `index.html` 클릭
2. 연필 아이콘 (Edit) 클릭
3. 수정 후 `Commit changes` 클릭
4. 몇 분 후 자동으로 업데이트됩니다
